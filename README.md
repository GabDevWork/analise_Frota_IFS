# 🚚 Dashboard de Business Intelligence: Análise de Frota

## 📌 Sobre o Projeto
Painel interativo desenvolvido em Power BI (`Analise_Frota_IFS.pbix`) focado na visualização gerencial de indicadores de frota. O projeto envolveu desde a modelagem dos dados via Power Query até a construção de métricas complexas em DAX, resultando em uma interface gráfica voltada para a tomada de decisão. *(Imagens do dashboard disponíveis no repositório).*

## 🔬 Relação com o Processo KDD
Este projeto atende diretamente à etapa final do KDD: **Interpretação e Avaliação dos Resultados**. Na descoberta baseada em bases de dados, padrões e agrupamentos precisam ser traduzidos visualmente. Este dashboard comprova a habilidade de:
* Utilizar ferramentas de Business Intelligence (BI) para facilitar o entendimento dos resultados.
* Mapear conexões implícitas e explícitas de forma gráfica.
* Traduzir matrizes de dados complexas em relatórios visuais que suportam a avaliação de tendências e hipóteses.

## 🛠️ Tecnologias Utilizadas
* **Microsoft Power BI** (Desenvolvimento do Dashboard)
* **Power Query** (Modelagem e processo de ETL)
* **Linguagem DAX** (Criação de cálculos e métricas avançadas)
